## üöÄ How to Run Locally

To explore or test any of these projects locally, clone this repository to your machine:

```bash
git clone https://github.com/pamodmalith/roadmap-backend-projects.git
cd roadmap-backend-projects
```

Then, navigate to the specific project folder you want to run. Each project contains its own isolated environment. For example:

```bash
cd task-tracker-cli
npm install
npm run build
npm start
```

(Note: Check the individual README.md inside each project folder for specific setup instructions and command usage.)

This repository is continuously updated as I progress through the backend roadmap.
